# Tele-Fundus 仕様書・設計ドキュメント

## 1. システム概要
遠隔眼底画像診断支援プラットフォーム「Tele-Fundus」は、健診センターからの症例登録、専門医による読影、オペレーターによるQC、および複雑なボリュームディスカウントを伴う会計処理までを単一リポジトリで一気通貫に担う基幹プラットフォームです。

## 2. アーキテクチャ
- **Backend**: Hono + TypeScript (Node.js環境で動作)
- **Frontend**: React + Vite + TypeScript (Tailwind CSS, shadcn/ui)
- **Database**: PostgreSQL 15 (UUID主キー、JSONB活用)
- **Infrastructure**: Docker Compose (ローカル開発用)

## 3. マルチエージェント・オーケストレーション
システムは「RootAgent」を司令塔とするマルチエージェント構造で動作し、ToolContextのstateを通じて情報を共有します。

| エージェント | 主な役割 |
|---|---|
| RootAgent | 司令塔。ユーザー要求を理解しエージェントを制御。 |
| ClientAgent | 依頼元入力支援。症例・画像登録。 |
| AssignmentAgent | ディスパッチャー。適切な医師への割当。 |
| ReadingAgent | 仮想読影医。画像分析と所見作成。 |
| QCAgent | 品質保証。医学的妥当性のチェック。 |
| BillingAgent | 会計担当。請求・支払額の正確な算定。 |

## 4. データベース設計原則
- **症例中心アクセス制御**: `screening_id` を軸に、全関連データ（画像、所見、チャット）の参照保全を担保。
- **データ整合性**: 空メッセージの禁止、症例削除時のカスケード削除の明示。

## 5. 会計ロジック (Core Billing Engine)
### 5.1 請求 (Client Billing)
- 月間階層別料金:
  - 0-500件: 1,000円/件
  - 501-1,000件: 900円/件
  - 1,001件以上: 800円/件
- オプション: 至急(+500円)、AI解析(+200円)

### 5.2 支払 (Physician Payout)
- ランク単価: 指導医(600円)、専門医(500円)、一般医(400円)
- 条件: 深夜(22-5時) +20%増し。見落とし率5%超で報酬調整。
